# Sales and Demand Forecasting

This project predicts future sales using historical data with machine learning techniques.

## Project Description
The goal of this project is to analyze past sales data and forecast future demand to help businesses make better decisions.

## Files
- sales dataset.csv → Dataset used for training
- sales and demand forecasting.ipynb → Main forecasting notebook

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Outcome
The model predicts future sales trends based on historical data.
